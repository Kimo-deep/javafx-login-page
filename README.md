# Login Application (JavaFX)

## 📌 وصف المشروع
هذا المشروع عبارة عن واجهة تسجيل دخول بسيطة باستخدام مكتبة **JavaFX**.  
الواجهة تحتوي على:
- حقل إدخال لاسم المستخدم.
- حقل إدخال لكلمة المرور.
- مربع اختيار (CheckBox) لتفعيل خيار "تذكرني".
- زر تسجيل الدخول (Login).
- تصميم مخصص للألوان والخطوط والخلفية.

---

## 🛠️ المتطلبات
لتشغيل المشروع تحتاج إلى:
- **Java JDK 8 أو أحدث**.
- **JavaFX SDK** (إذا لم يكن مدمجًا مع JDK).
- بيئة تطوير مثل **IntelliJ IDEA** أو **Eclipse** أو يمكنك تشغيله عبر سطر الأوامر.

---

## ▶️ كيفية التشغيل
1. انسخ الكود داخل ملف باسم `login.java` داخل حزمة `t`.
2. تأكد من إعداد مكتبة JavaFX في مشروعك.
3. شغّل البرنامج باستخدام الأمر:

```bash
javac t/login.java
java t.login
